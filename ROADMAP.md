# CguP Roadmap

Our vision is to build a "Concurrent Go Rendering Engine".

## Phase 1: Foundation (Current)
- [x] Basic Architecture Setup
- [x] Window Creation (GLFW)
- [x] Vulkan Instance & Device Selection
- [ ] Cross-platform support verification (Linux/Windows)

## Phase 2: The Triangle
- [ ] Swapchain Creation
- [ ] Graphics Pipeline
- [ ] Shader Compilation (SPIR-V)
- [ ] Command Buffers & Rendering

## Phase 3: The Engine
- [ ] Render Graph functionality
- [ ] Multithreaded Command Recording
- [ ] Asset System (glTF loading)

## Phase 4: Advanced Features
- [ ] PBR Materials
- [ ] ECS Integration
- [ ] Scripting Layer (maybe Lua or Go plugins)
