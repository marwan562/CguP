# Contributing to CguP

Thank you for your interest in contributing to CguP! We are building a modern, concurrent rendering engine in Go.

## Code of Conduct

Please be respectful and inclusive. We are all here to learn.

## How to Contribute

1.  **Fork the repo** and create your branch from `main`.
2.  **Ensure you have the Vulkan SDK installed**.
3.  **Run tests**: `go test ./...`
4.  **Submit a Pull Request**.

## Style Guide

- Use `gofmt` (standard Go formatting).
- Document exported functions.
- Keep concurrency logic in `engine/compute` or clear separate modules.

## "Good First Issues"

Check the [Issues](https://github.com/marwan562/CguP/issues) tab for tasks labeled `good first issue`.
